# Thulir ERP System - Construction Management

A comprehensive ERP software for a construction company to manage workers, attendance, sites, and payments.

## Features

*   **Worker Management:** Register and manage workers, including their roles and wage types (Daily/Piece Work).
*   **Site Management:** Create and manage construction sites with geofencing capabilities.
*   **Attendance Tracking:**
    *   Location-locked punch-in/out for workers.
    *   Manual attendance corrections by owners.
    *   Real-time status updates (Present, Absent, Half Day).
*   **Team Management:** Organize workers into teams and assign representatives.
*   **Advance Payments:** Track advance payments given to workers.
*   **Dashboard:**
    *   Owner's dashboard for overview and management.
    *   Weekly reports for attendance and payments.

## Tech Stack

*   **Frontend:** React, TypeScript, Vite
*   **Styling:** Tailwind CSS (via CDN/index.css methodology) or Standard CSS
*   **Icons:** Lucide React
*   **Date Handling:** date-fns
*   **Router:** React Router DOM
